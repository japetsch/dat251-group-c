# See closest location

As a blood donor,
I want to see the closest donation location to me,
so that i can quickly find the most convenient place to donate.

## Acceptance criteria

- User can trigger "find closest location"
- App request location permission
- If permission is granted, show the closest donation location based on the user's current location
- Display distance
- If permission is denied or location is unavailable, allow entering an address/postcode to calculate the closest location
