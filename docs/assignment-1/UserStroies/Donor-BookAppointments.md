# Book appointments

As a blood donor (registered user),
I want to be able to book an appointment,
so that I can donate blood.

## Acceptance criteria

- Book free/available timeslots
- Get error when timeslot was booked in the meantime
- Get error when one of the donation-requirements is not met:
    - Verify no previous donation in the last 3-4 months
    - Verify exclusion rules (sickness, cuts, etc.)
