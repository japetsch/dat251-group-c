# Change user data

As a blood donor,
i want to be able to view and update my personal details,
so that my contact and address information stays correct.

## Acceptance criteria

- User can view current profile data
- User can edit and save changes to profile data
- Validate input before saving:
    - Email must be a valid format
    - Phone number must be a valid format
    - Required fields cannot be empty
- Show a success message after saving changes
- Show an error message if saving fails
